# Cross-Cutting Reports

Synthesis and evidence reports that span all thematic buckets. These provide the big-picture view of OCL's 7+ year issue history.

## Reading Order

1. **[phase-1-summary.md](phase-1-summary.md)** — How the 633 open issues were bucketed into thematic groups for analysis
2. **[historical-backlog-synthesis.md](historical-backlog-synthesis.md)** — The 510-issue cross-bucket synthesis organized by 8 themes (TermBrowser v3, API, UX/UI, Infrastructure, Community, DevEx, Mapper, CIEL). This is the most comprehensive single document.
3. **[code-evidence.md](code-evidence.md)** — Commit scanning of 511 issues against 6 repos. Identifies which issues have actual code activity (69 with evidence, 442 without).
4. **[active-review.md](active-review.md)** — The 48 issues that required human review during Phase 1
5. **[post-review-summary.md](post-review-summary.md)** — Executive summary of Phase 1 outcomes: what was kept, closed, moved to active, or moved to backlog
